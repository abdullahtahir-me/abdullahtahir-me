## 🚀 Projects

### [Calenme](https://github.com/abdullahtahir-me/calenme) — Full-Stack Student Planner
> Live at [calenme.vercel.app](https://calenme.vercel.app)

A full-stack academic planner for university students. Built with Next.js App Router and Supabase, with server-side Google OAuth, Row-Level Security on all user data, and a custom PostgreSQL function that wraps course + schedule creation in an atomic transaction.

`Next.js` `TypeScript` `Supabase` `PostgreSQL` `Tailwind CSS` `Vercel`

---

### [ProjectT](https://github.com/abdullahtahir-me/ProjectT) — 2D Destructible Terrain Game
A physics-based 2D game inspired by *Worms* and *Terraria*. Features real-time terrain destruction via pixel manipulation, projectile kinematics, and heightmap-based collision detection.

`Java` `libGDX` `Gradle`

---

### [PingPong](https://github.com/abdullahtahir-me/PingPong) — Classic Pong in C
A two-player Pong game built from scratch using C and Raylib. Includes a full game loop, multiple screens (loading, menu, gameplay, game over), real-time collision detection, and progressive difficulty.

`C` `Raylib`

---

### [Emotis](https://github.com/abdullahtahir-me/Emotis-Mood-Sharing) — Real-Time Mood Sharing
A minimal two-person mood-sharing app. Users connect via a shared PIN and see each other's emotional state update in real time across browser tabs.

`React` `TypeScript` `Vite` `Tailwind CSS`

---

### [img2ascii](https://github.com/abdullahtahir-me/img2ascii) — Image to ASCII Art
Converts any image to ASCII art in the terminal by mapping per-pixel grayscale brightness to a custom character scale. Pure Java, no external dependencies.

`Java`
